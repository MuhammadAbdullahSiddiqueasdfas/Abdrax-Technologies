# Application Architecture

This project is a modern web application for **Abdrax Technologies**, a software house. It is built with high performance, SEO, and premium design in mind.

## Core Technology Stack

- **Framework**: [Next.js](https://nextjs.org/) (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion & CSS Transitions
- **Icons**: Lucide React & React Icons
- **Components**: shadcn/ui (customized)

## Design Principles

- **Dark Mode First**: The site is designed with a sleek, dark aesthetic.
- **Glassmorphism**: Use of semi-transparent backgrounds with backdrop filters.
- **Micro-animations**: Subtle transitions and hover effects to enhance UX.
- **Responsiveness**: Fully optimized for mobile, tablet, and desktop.

## State Management

- Primarily uses React Hooks (`useState`, `useEffect`) for UI state.
- Next.js App Router handles server-side state and navigation.
